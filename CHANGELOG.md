# Changelog

All notable changes to this project are documented here.
This project adheres to [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2026-06-14
### Added
- Initial public release.
- Single-Worker PipOS shell with parent/child mode and PIN gate.
- De-stimulation engine: grayscale/contrast/brightness/warm-tone/saturation filter overlay.
- Three presets (Calm / Focus / Sleep) + manual sliders.
- YouTube search & playback with **no API key** (InnerTube proxy + youtube-nocookie embeds).
- Local file, HTTP stream, and NAS/SMB playback paths.
- KV-backed per-device settings.
